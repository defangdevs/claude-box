▣ claude-box

Reproducible, multi-user Claude Code agent sandboxes — one click on AWS, on bare metal, or as a VM image.

Launch on AWS View on GitHub

What it is

A ready-to-use Claude Code agent you open right in your browser. Set it up once with a single click; it keeps running on its own.

Locked down by default

Each agent runs in its own restricted account and can't touch the rest of the machine — it only gets the specific extra powers you choose to grant.

Works in your browser

A full terminal, right in your browser over a secure connection. Open your link and you're in — nothing to install.

Keeps your keys safe

Any API keys or tokens you add stay private — kept out of shared settings and never baked into saved images.

Low cost

Runs lean and skips an optional AWS network fee by default, so a small box typically costs only a few dollars a month. Shut it down anytime to stop charges.

1-click AWS launch

Sets up one ready-to-go box in your own AWS account with the browser terminal already running. It creates everything it needs, so there's nothing to configure beforehand — just pick a region and go.

You don't need to make up a password. Every Launch button already has a random one filled in for you. Keep it, type your own, or clear it to have one generated automatically. Pick a size and launch. The box takes about 2–3 minutes to start. When it's ready, your private link (with the password already built in) appears under Outputs in the AWS console — open it, finish the one-time Claude sign-in, and you're in.
If your link won't open: the box uses a modern, free type of internet address that most home connections support, but some office and public Wi-Fi networks can't reach it. If your link never loads, launch again and set PublicIpv4 to true — that adds an address every network can reach, for about $3.60/month.

Other ways to run

The AWS template just imports the same NixOS module you'd use anywhere.

Bare metal / existing NixOS host

{
  imports = [ inputs.claude-box.nixosModules.claude-box ];

  services.claude-box = {
    enable = true;
    users = {
      alice = { };
      bob   = { remoteControlName = "bob-box"; };
    };
    # The ONLY elevated powers the agents get -- keep it tight.
    sudoAllowlist = [ "/run/current-system/sw/bin/systemctl reload caddy.service" ];
    extraPackages = with pkgs; [ git ripgrep jq ];
  };
}
