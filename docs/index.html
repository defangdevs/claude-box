▣ claude-box

Reproducible, multi-user Claude Code agent sandboxes for NixOS — on bare metal, as a VM image, or one click on AWS.

Launch on AWS View on GitHub

What it is

Each agent is an unprivileged user running Claude Code in a persistent tmux session, defined once in declarative config.

Unprivileged by default

Agents run as ordinary users. The only elevated power is a tight, explicit passwordless-sudo allowlist you control.

Browser terminal

Caddy + ttyd serve the tmux session over HTTPS with an auto-issued TLS cert. Open a URL, you're in.

Secret-safe token injection

Custom tokens (e.g. GH_TOKEN) arrive via drop-in EnvironmentFiles that never enter the world-readable Nix store.

Cheap by design

IPv6-only by default to skip AWS's ~$3.60/mo public-IPv4 charge, with an opt-in Spot toggle for ~70–90% off compute.

1-click AWS launch

Provisions one EC2 instance (NixOS 25.11) with the module and browser terminal already wired up. It builds its own IPv6-enabled VPC, so nothing on your account needs pre-configuring.

Set a WebPassword (16+ URL-safe chars — it becomes a path token in the URL, since browsers don't reliably attach Basic Auth to WebSocket upgrades), pick a size, and launch. First load takes ~2–3 min while the box provisions and Caddy issues a cert. The stack Outputs show https://<addr>.sslip.io/<token>/ — open it, finish the one-time Claude sign-in, done.
IPv6-only by default. Works if your client has IPv6 (most consumer ISPs in NA/EU). On a network without it, set PublicIpv4: true at launch to allocate an Elastic IP (adds ~$3.60/mo, works everywhere).

Other ways to run

The AWS template just imports the same NixOS module you'd use anywhere.

Bare metal / existing NixOS host

{
  imports = [ inputs.claude-box.nixosModules.claude-box ];

  services.claude-box = {
    enable = true;
    users = {
      alice = { };
      bob   = { remoteControlName = "bob-box"; };
    };
    # The ONLY elevated powers the agents get -- keep it tight.
    sudoAllowlist = [ "/run/current-system/sw/bin/systemctl reload caddy.service" ];
    extraPackages = with pkgs; [ git ripgrep jq ];
  };
}
