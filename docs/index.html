▣ claude-box

A private, sandboxed Claude Code or Codex agent for you and your team — launch it in your own AWS account with one click.

Launch on AWS View on GitHub

What it is

A ready-to-use coding agent you open right in your browser. Set it up once with a single click; it keeps running on its own.

Locked down by default

Each agent runs in its own restricted account and can't touch the rest of the machine — it only gets the specific extra powers you choose to grant.

Works in your browser

A full terminal, right in your browser over a secure connection. Open your link and you're in — nothing to install.

Choose your agent

Pick Claude Code or Codex when you launch. The same NixOS module, terminal, and sandboxing model support both.

Keeps your keys safe

Any API keys or tokens you add stay private — kept out of shared settings and never baked into saved images.

Low cost

Runs lean and skips an optional AWS network fee by default, so a small box typically costs only a few dollars a month. Shut it down anytime to stop charges.

1-click AWS launch

Sets up one ready-to-go box in your own AWS account with the browser terminal already running. It creates everything it needs, so there's nothing to configure beforehand — just pick a region and go.

You don't need to make up a password. Every Launch button already has a random one filled in for you. Keep it, type your own, or clear it to have one generated automatically. Choose Claude Code or Codex, pick a size, rename the stack if you want a friendlier Claude app label, and launch. The box takes about 2–3 minutes to start. When it's ready, your browser link appears under Outputs in the AWS console — open it, enter the password from the deployment form, finish the selected agent's one-time sign-in, and you're in.
Modern internet address by default. The box uses a free, IPv6 address that most home connections support, but some office and public Wi-Fi networks can't reach it. We check your browser on this page — if we can't reach the IPv6 internet from here, the Launch button above will automatically switch to IPv4 (~$3.60/month) so the box will actually be reachable.

Other ways to run

If your team already has its own server, or prefers a downloadable virtual-machine image, the same setup is available for your IT team to install.
